¿Qué es este proyecto?

Este proyecto tiene la intención de aprovechar más el agua lluvia, creando un sistema que permita recolectarla, para luego purificarla, y usarla para diferentes fines, como lo son: lavar platos, lavar ropa, lavar la trapeadora etc. Este sistema de purificación funciona mediante filtros, y todo esto se hace de forma automática. Sin embargo, una de las intenciones de este proyecto es incentivar a las personas para que utilicen el agua lluvia, y por esto pensamos en enlazar este sistema con la tecnología del internet de las cosas, la cual permite tener una conexión entre el sistema en si y una red de internet, por lo que una persona podría obtener diferente información de su sistema de recolección desde un computador o desde su celular.

El desarrollo del sistema de recolección de agua lluvia, se dividió en dos partes, las cuales son: los filtros y el hardware y software

Se desarrollaron 3 filtros que se encargan de purificar el agua, estos se uniran al tanque que recoge el agua luvia, para permitir que el proceso de purificacion sea de forma automatica

Este primer filtro funciona de una manera muy básica, ya que el objetivo de este es que funcione como colador para que se puedan separar las partículas más grandes del agua, y de esta forma evitar posibles obstrucciones en las tuberías, canillas y cuidar un poco los otros filtros, para lo cual se colocará una maya que simplemente permitiera que el agua pase
Este filtro ya es de purificación en sí, y funciona con carbón activado, que es una sustancia muy utilizada en la purificación de diferentes líquidos como el agua, esto gracias a su capacidad de absorción de sustancias, principalmente orgánicas, entonces, el agua al ir pasando por allí, se va purificando
Este filtro es en sí un filtro de seguridad, para estar más seguro de que el agua lluvia este completamente purificada, para esto se usa ozono, el cual también es descontaminante como el carbón activado, sin embargo, este es más fuerte para la eliminación de sustancias orgánicas, y por esto se usa como filtro de seguridad

Para poder conectar todo este sistema con un celular y/o el computador, se usó un módulo wifi ESP8266, que es un pequeño modulo que lo que hace en si es abrir una IP, en la red wifi que está conectado, y luego envía información del arduino a esta IP, por lo que los dispositivos que estén conectados a esta IP podrán ver esa información o incluso enviarle ordenes al arduino, y el requisito del dispositivo para conectarse a esa IP es estar conectado a la misma red wifi, que está conectado el modulo
